A knock knock joke server.

## To Run

### Activate virtualenv

From this directory:

`source bin/activate`

### Run Server

`python3 knock_knock_server.py`

### Exit Server

`CTRL-C` will exit the server gracefully, waiting for all connections to terminate.

### Connect to Server

`telnet <cluster> 47684`
